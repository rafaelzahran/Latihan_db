🚀 Fitur Utama
1. Struktur Dokumen (HTML)
Heading Bertingkat: Menggunakan tag <h1> sampai <h6> untuk menunjukkan hierarki konten.

Media: Menampilkan gambar (cilok/makanan) sebagai elemen visual utama.

Sidebar & Artikel: Menggunakan elemen semantik <aside> untuk informasi tambahan dan tag paragraf untuk konten artikel.

Formulir Input: Dilengkapi dengan berbagai tipe input seperti teks, nomor, tanggal, waktu, hingga pemilih warna (color picker).

Tabel Data: Menampilkan informasi terstruktur (Nama dan Umur) menggunakan tag <table>.

2. Penataan Gaya (CSS)
Typography: Menggunakan font sans-serif untuk tampilan yang bersih.

Visual Sidebar: Memberikan latar belakang abu-abu dan border kiri sebagai penanda area informasi tambahan.

Tabel: Pengaturan border-collapse dan padding agar data di dalam tabel mudah dibaca.

Warna Kustom: Implementasi class khusus untuk pewarnaan teks tertentu (merah dan orange).

3. Logika Interaktif (JavaScript)
Manipulasi DOM: Mengisi data mahasiswa secara otomatis ke dalam elemen HTML saat halaman dimuat.

Event Listeners: * Alert: Memunculkan pesan pop-up saat tombol diklik.

Style Switcher: Mengubah teks judul dan warnanya secara dinamis melalui tombol "Ubah Teks".

Input Handler: Menangkap input nama dari pengguna dan menampilkannya sebagai sapaan di layar, lengkap dengan validasi jika input kosong.

🛠️ Teknologi yang Digunakan
HTML5 (Struktur)

CSS3 (Desain & Layout)

JavaScript (Interaksi)
